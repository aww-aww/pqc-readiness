# pqc-readiness
Stub out the basic framework for a set of PQC readiness tracking pages to be hosted under the Post-Quantum Cryptography Alliance (PQCA) repo

The official location for this information is: https://github.com/PQCA/wg-readiness-tracking

Open questions:

* What license should be used? (Settled - PQCA standard is the Apache license and it will be used for the public repo)
* How do we carve up the space? Proposed: Cryptography Libraries / Tools, Cloud Services, HSMs, Networking Devices, Operating Systems, Software
